# Analog Clock – Simple Summary

Author: **Irem Dönmez**  
Purpose: A responsive analog clock made with CSS and JavaScript.

---

## CSS (Design)

**Global settings**
  - Removes default margins and padding.
  - Makes all elements use `border-box`.

**Body**
  - Centers the clock on the screen.
  - Dark red background.
  - Uses Arial font.

**Clock container**
  - The clock is 300×300px (gets smaller on small screens).

**Clock face**
  - Outer circle with a thick border.
  - Inner circle that holds the hands.

**Hour marks**
  - Four marks at 12, 3, 6, and 9 o’clock.
  - Each mark is a small rectangle, rotated into position.

**Clock hands**
  - All hands rotate from the right side.
  - **Hour hand:** thicker, black.
  - **Minute hand:** gray.
  - **Second hand:** thin and red.

**Center circle**
  - A small circle that covers the base of the hands.

**Responsive**
  - On screens under 400px: clock becomes 200×200px.
